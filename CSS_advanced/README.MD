# CSS Advanced - Techium Project

## Description

This project focuses on applying CSS styling to the Techium webpage built in the HTML Advanced project.
The goal is to learn CSS fundamentals and progressively build a fully styled webpage.

## Learning Objectives

By the end of this project, you should be able to explain:

- What is CSS
- How to add style to an element
- What is a class
- What is a selector
- How to compute CSS Specificity Value
- What are Box properties in CSS
- How does the browser load a webpage

## Requirements

- All files are interpreted on Ubuntu 20.04 LTS
- Editors allowed: `vi`, `vim`, `emacs`
- All files should end with a new line
- All files should start with a comment describing the task
- Code should be W3C compliant and validate with W3C-Validator (when specified)
