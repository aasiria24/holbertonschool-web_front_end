# HTML Advanced - Techium Project

This directory contains the structured HTML files for the Techium website, built as part of the Holberton School curriculum. The focus is on **semantic HTML5**, proper document structure, and accessibility—no CSS or styling is applied.

## Files included

- `0-index.html` … `39-index.html` : Step‑by‑step construction of the main homepage.
- `index.html` : Final, complete homepage (with logo and SVG icons).
- `about.html`, `latest_news.html`, `contact.html` : Variants of the homepage with updated `<title>`.
- `article.html` : Example of an article page (with `<aside>`).
- `styleguide.html` : Comprehensive style guide showing headings, lists, tables, media, iframes, etc.
- `*-styleguide.html` : Intermediate styleguide files (e.g., `11-styleguide.html` … `39-styleguide.html`).

All files are **W3C compliant** (except where noted in tasks) and demonstrate the use of:
- Semantic tags (`<header>`, `<main>`, `<footer>`, `<section>`, `<article>`, `<aside>`, `<nav>`)
- Proper heading hierarchy (`<h1>` – `<h6>`)
- Lists (`<ul>`, `<ol>`, `<dl>`)
- Tables with `<caption>` and `scope`
- Media: `<video>`, `<audio>`, `<iframe>`, `<img>` with `alt` attributes
- Forms, blockquotes, `<details>`, `<mark>`, and more.
